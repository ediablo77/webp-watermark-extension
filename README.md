# WebP Watermark Extension

اکستنشن Google Chrome برای ذخیره تصاویر هر وب‌سایتی با فرمت **WebP** همراه با قابلیت افزودن **واترمارک سفارشی**.

---

## ✨ امکانات

- ذخیره هر تصویر با فرمت WebP از طریق راست‌کلیک
- تبدیل تصاویر JPG، PNG، BMP، GIF و سایر فرمت‌ها به WebP
- افزودن واترمارک متنی دلخواه
- انتخاب محل نمایش واترمارک:
  - بالا چپ
  - بالا راست
  - وسط تصویر
  - پایین چپ
  - پایین راست
- تنظیم میزان شفافیت (Opacity) واترمارک
- تنظیم کیفیت خروجی WebP
- حجم کمتر با حفظ کیفیت مناسب تصویر
- عملکرد کاملاً آفلاین
- بدون ارسال اطلاعات به سرورهای خارجی

---

## 📥 نصب افزونه در Google Chrome

1. پروژه را دانلود یا Clone کنید.
2. در صورت دانلود فایل ZIP، آن را از حالت فشرده خارج کنید.
3. مرورگر Chrome را باز کنید و وارد آدرس زیر شوید:

```text
chrome://extensions
```

4. گزینه **Developer Mode** را فعال کنید.
5. روی **Load Unpacked** کلیک کنید.
6. پوشه استخراج‌شده افزونه را انتخاب کنید.

اکستنشن با موفقیت نصب خواهد شد.

---

## ⚙️ تنظیمات افزونه

پس از نصب:

1. روی آیکون افزونه در نوار ابزار Chrome کلیک کنید.
2. متن واترمارک موردنظر را وارد کنید.
3. محل نمایش واترمارک را انتخاب کنید.
4. میزان شفافیت (Opacity) را تنظیم کنید.
5. کیفیت خروجی WebP را مشخص کنید.
6. روی دکمه **Save** کلیک کنید.

تنظیمات به‌صورت خودکار ذخیره می‌شوند.

---

## 🚀 نحوه استفاده

1. وارد هر وب‌سایتی شوید.
2. روی تصویر موردنظر راست‌کلیک کنید.
3. گزینه زیر را انتخاب کنید:

```text
Save image as WebP
```

4. افزونه به‌صورت خودکار:
   - تصویر را به فرمت WebP تبدیل می‌کند.
   - واترمارک انتخاب‌شده را روی تصویر قرار می‌دهد.
   - فایل نهایی را دانلود می‌کند.
