# webp-watermark-extension
امکانات نسخه نهایی راست‌کلیک روی هر تصویر → Save image as WebP تبدیل هر فرمتی به WebP تنظیم: متن واترمارک محل واترمارک: بالا چپ بالا راست وسط پایین چپ پایین راست شفافیت (Opacity) کیفیت خروجی (Quality) کیفیت خوب + حجم کمتر کاملاً آفلاین (بدون سرور)
